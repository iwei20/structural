# structural
Implementation of Abel and Altenkirch's foetus termination checker

Call by using Structural.Call.check from `lib/check.ml`. The calculus terms
are as in `lib/calculus.ml`.

## References
A. ABEL and T. ALTENKIRCH, "A predicative analysis of structural recursion,"
Journal of Functional Programming, vol. 12, no. 1, pp. 1–41, 2002.
doi:10.1017/S0956796801004191
